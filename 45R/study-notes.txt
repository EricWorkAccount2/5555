Source Study Notes for 45R/src

Top-level files:
- src/main.tsx
  - Imports: App, UserProvider, '@/index.css', StrictMode, createRoot, BrowserRouter.
  - Export: none.
  - Purpose: application bootstrap, mount root React tree within BrowserRouter and UserProvider.

- src/App.tsx
  - Imports: MainLayout, all view components through '@/views', TVView, Route, Routes.
  - Export: App component.
  - Purpose: defines React Router routes and nested routes for movies, TV, people, favorites, cart, settings, and genres.

- src/index.css
  - Imports: none in source file.
  - Export: none.
  - Purpose: global CSS used across the app.

Shared barrel files:
- src/components/index.ts
  - Re-exports: Button, ButtonGroup, ImageGrid, ImageOverlay, Link, LinkGroup, Pagination, SearchBar, DetailItem, Header, Modal.
  - Purpose: one import point for shared components.

- src/context/index.ts
  - Re-exports: UserContext, UserProvider.
  - Purpose: one import point for context.

- src/core/index.ts
  - Re-exports: constants and utilities from core/constants and core/Utility.
  - Purpose: shared core entry point for constants, types, and helper functions.

- src/views/index.ts
  - Re-exports: all view components used by App.
  - Purpose: simplifies view imports in App.tsx.

Component library:
- src/components/controls/buttons/Button.tsx
  - Imports: ReactNode.
  - Consts: baseStyles, variants.
  - Export: Button.
  - Purpose: reusable button with variant styling and disabled state.

- src/components/controls/buttons/ButtonGroup.tsx

- src/components/controls/images/ImageGrid.tsx

- src/components/controls/images/ImageOverlay.tsx
  - Imports: ImageAction, ImageCell types.
  - Export: ImageOverlay.
  - Purpose: renders action buttons over an image card and stops click propagation.

- src/components/controls/links/Link.tsx
  - Imports: ReactNode, matchPath, NavLink, useLocation.
  - Export: Link.

- src/components/controls/links/LinkGroup.tsx
  - Imports: Link from '@/components'.

  - Purpose: renders a horizontal group of navigation links.

- src/components/controls/Pagination.tsx
  - Imports: Button from '@/components'.
  - Consts: safeMaxPages, goPrev, goNext.
  - Export: Pagination.
  - Purpose: renders pagination controls and handles page increment/decrement.

- src/components/controls/SearchBar.tsx
  - Imports: none.
  - Export: SearchBar.
  - Purpose: search input component with change callback.

- src/components/site/DetailItem.tsx
  - Imports: none.
  - Export: DetailItem.
  - Purpose: small label-value detail card used in detail modals.

- src/components/site/Header.tsx
  - Imports: Link from '@/components', ICON_SIZE from '@/core', useUserContext, FaRegHeart, FaShoppingCart, GoGear, NavLink.

  - Purpose: top navigation bar with page links, user name, favorites/cart counters, and settings link.

- src/components/site/Modal.tsx
  - Imports: ReactNode.
  - Export: Modal.
  - Purpose: modal overlay wrapper that closes on backdrop click.

Application context:
- src/context/UserContext.ts
  - Imports: ImageCell type.
  - Exports: UserContext and UserContextType.
  - Purpose: type definitions for username, favorites, cart, and toggle handlers.

- src/context/UserProvider.tsx
  - Imports: ImageCell type, storage keys (CART_KEY, FAVORITES_KEY, USERNAME_KEY), useLocalStorage, ReactNode, UserContext.
  - Consts: userName, favorites, cart state hooks.
  - Functions: toggleFavorite and toggleCart.
  - Export: UserProvider.
  - Purpose: persists user settings and collections in localStorage and ensures cart/favorites are exclusive.

Core shared utilities and types:
- src/core/constants/components.ts
  - Exports: RATE_LIMIT_DELAY = 500.

Custom hooks:

Notes:
- Most views use useTmdb for API data fetching and display a loading message while waiting.
- Card components use ImageCell objects created from API results and pass those into ImageGrid.
- Cart and favorites are managed through UserProvider and are mutually exclusive in state.
- Route navigation is handled by React Router with nested child routes inside App.tsx.
